# MVP - PUC-Rio - Medical Appointment No Shows

Este repositório contém o MVP da Sprint 1 da disciplina **Análise de Dados e Boas Práticas** da **PUC-Rio**.

O projeto foi desenvolvido por **Bernardo C. M. A. Pereira** e tem como objetivo analisar o problema de **não comparecimento a consultas médicas** a partir do dataset **Medical Appointment No Shows**, cobrindo as etapas de:

- definição do problema;
- análise exploratória de dados;
- pré-processamento de dados.

O trabalho foi estruturado com base no template disponibilizado pela disciplina e busca seguir as boas práticas discutidas ao longo da sprint.

## Objetivo do projeto

O objetivo deste MVP é investigar quais características dos pacientes e dos agendamentos parecem estar associadas ao não comparecimento às consultas médicas, utilizando uma abordagem orientada por dados desde a formulação do problema até a preparação da base para etapas futuras.

## Principais etapas desenvolvidas

- definição do problema e formulação de hipóteses;
- descrição e inspeção inicial do dataset;
- análise exploratória com estatísticas descritivas e visualizações;
- identificação de inconsistências e padrões relevantes;
- tratamento e preparação dos dados;
- reavaliação das hipóteses com base nos resultados obtidos.

## Principais achados

Entre os principais resultados observados no projeto, destacam-se:

- uma forte relação entre **tempo de espera** e taxa de não comparecimento;
- diferenças de ausência entre **faixas etárias**;
- variações nas taxas de falta entre **bairros**;
- presença de inconsistências que exigiram tratamento, como idades e tempos de espera inválidos.
